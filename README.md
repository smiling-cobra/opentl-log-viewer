# OTLP Log Viewer

Web app for visualizing OTLP log records — flat list, histogram, and grouped-by-service views.

## Prerequisites

- Node.js 20

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the dev server:
   ```sh
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000).
